# AWS Resource Access Manager (AWS RAM)

## What is AWS RAM?

AWS Resource Access Manager (RAM) enables secure sharing of supported AWS resources across AWS accounts and AWS Organizations.

## Components

- Resource Shares
- Shared Resources
- Principals
- Managed Permissions
- AWS Organizations

## Benefits

- Centralized resource management
- Reduced infrastructure duplication
- Simplified multi-account networking
- Improved governance
- Better operational efficiency

## Resources That Can Be Shared

- Transit Gateway
- VPC Subnets
- Prefix Lists
- Route 53 Resolver Rules
- Network Firewall Policies
- EC2 Capacity Reservations

## Enterprise Use Cases

- Landing Zones
- Shared Services
- Multi-account networking
- Centralized networking
- Enterprise governance

## Best Practices

- Use AWS Organizations
- Apply least-privilege permissions
- Monitor sharing with CloudTrail
- Audit resource shares regularly
