# Amazon VPC IPv6 Networking

## What is IPv6 Networking?

Amazon VPC IPv6 Networking enables AWS resources to communicate using IPv6 addresses in standalone or dual-stack deployments.

## Components

- IPv6 CIDR Block
- Dual-Stack VPC
- IPv6 Subnets
- Route Tables
- Internet Gateway
- Egress-Only Internet Gateway
- Security Groups
- Network ACLs

## Benefits

- Larger address space
- Improved scalability
- Native internet connectivity
- Future-ready networking

## Enterprise Use Cases

- Dual-stack deployments
- Hybrid cloud
- Global applications
- Enterprise IPv6 adoption

## Best Practices

- Use dual-stack where possible
- Plan IPv6 CIDR allocations
- Secure IPv6 traffic with Security Groups and NACLs
- Use Egress-Only Internet Gateway for private workloads
- Monitor traffic with VPC Flow Logs
