# Amazon Route 53 Resolver DNS Firewall

## What is Route 53 Resolver DNS Firewall?

Amazon Route 53 Resolver DNS Firewall helps protect AWS workloads by filtering outbound DNS queries to malicious or unauthorized domains.

## Benefits

- DNS-layer security
- Malware protection
- Phishing prevention
- Centralized DNS filtering
- Easy VPC integration

## Components

### Rule Groups

Logical collections of DNS filtering rules.

### Domain Lists

Lists of domains that are allowed or blocked.

### Firewall Actions

- Allow
- Block
- Alert

### VPC Associations

Apply Rule Groups to one or more VPCs.

## Enterprise Use Cases

- DNS security
- Zero Trust architecture
- Compliance
- Threat prevention
- Secure hybrid cloud

## Best Practices

- Maintain updated domain lists
- Use separate Rule Groups for different environments
- Enable query logging
- Monitor DNS activity regularly
- Review firewall policies periodically
