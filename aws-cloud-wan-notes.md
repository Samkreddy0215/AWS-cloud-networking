# AWS Cloud WAN

## What is AWS Cloud WAN?

AWS Cloud WAN is a managed wide area networking service that helps organizations build, manage, and monitor a global network across AWS Regions, branch offices, and data centers.

## Benefits

- Centralized network management
- Global connectivity
- Simplified routing
- Network segmentation
- Integration with AWS services

## Core Components

### Core Network

Provides global AWS backbone connectivity.

### Segments

Logical network separation.

Examples:

- Production
- Development
- Testing
- Security

### Network Policies

Control communication between segments.

## Connectivity Options

- Site-to-Site VPN
- AWS Direct Connect
- SD-WAN Integration
- Transit Gateway Integration

## Cloud WAN vs Transit Gateway

| Feature | Cloud WAN | Transit Gateway |
|----------|-----------|----------------|
| Global Connectivity | Yes | Limited |
| Regional Service | No | Yes |
| Segmentation | Advanced | Basic |
| Centralized Policies | Yes | Limited |

## Enterprise Use Cases

- Multi-Region Networking
- Global Branch Connectivity
- Hybrid Cloud Networking
- Network Segmentation
- Centralized Management

## Key Benefits

- Scalable architecture
