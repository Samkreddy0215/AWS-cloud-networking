# Amazon VPC Reachability Analyzer

## What is Reachability Analyzer?

Amazon VPC Reachability Analyzer verifies network connectivity between AWS resources by analyzing configuration paths without sending live network traffic.

## Components

- Source Resource
- Destination Resource
- Network Path
- Route Tables
- Security Groups
- Network ACLs

## Benefits

- Faster troubleshooting
- Configuration validation
- Network visibility
- Security verification
- Reduced downtime

## Supported Resources

- EC2
- ENI
- NAT Gateway
- Internet Gateway
- Transit Gateway
- VPC Peering
- Network Load Balancer

## Enterprise Use Cases

- Migration validation
- Production troubleshooting
- Change verification
- Security audits

## Best Practices

- Validate connectivity before deployments
- Review failed path explanations
- Combine with VPC Flow Logs
- Test after network changes
