# AWS Prefix Lists

## What are AWS Prefix Lists?

AWS Prefix Lists allow multiple CIDR blocks to be grouped into a reusable object for routing and security configurations.

## Types

### AWS Managed Prefix Lists

Maintained by AWS for AWS services.

### Customer-Managed Prefix Lists

Created and maintained by customers for their own network ranges.

## Benefits

- Centralized CIDR management
- Simplified Security Group rules
- Easier Route Table updates
- Improved scalability
- Consistent network policies

## Supported Resources

- Security Groups
- Route Tables
- Transit Gateway Route Tables

## Enterprise Use Cases

- Corporate office networks
- Hybrid cloud connectivity
- Multi-account networking
- Centralized security management

## Best Practices

- Use descriptive names
- Keep CIDR entries organized
- Review updates regularly
- Prefer AWS Managed Prefix Lists for AWS services
